# 🏋️ IRON 60 — Scheda Massa

**Scheda personalizzata full body 5gg** dedicata a utente di ~60kg con manubri 8kg e panca regolabile.  
30 min a sessione, massimo sforzo, set a cedimento.

## Utilizzo

Apri: **https://rosdade.github.io/iron60**

Include:
- Timer recupero integrato
- Tracker set con autocompletamento esercizi
- Modalità EMOM per il sabato
- Video dimostrativi per ogni esercizio
- Guida nutrizionale onboard
- Salvataggio automatico (localStorage)

## Tecnologia

Singola pagina HTML/CSS/JS vanilla. Nessun framework o dipendenza esterna.
